func ToJSON(r io.Reader) ([]byte, error)

ToJSON decodes 'r' containing an XML plist and returns the value as JSON-encoded bytes.

For example, the following XML: 

<array>
	<integer>1</integer>
	<string>foo</string>
</array>

is returned as `[1, "foo"]`.

Build version go1.16.15.
